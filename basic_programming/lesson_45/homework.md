# Task 1    
Написать метод, принимающий не пустой лист  целых чисел , в котором все числа кроме одного имеют пару и возвращающий
это не парное число.   При решении использовать мапу  
Примеры:  
1,3,1  -> 3    
3,3,3  -> 3  
1,1,15,1,1  -> 15  
1,8,8,8,8,8,1,2,2 -> 8
